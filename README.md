Local Weather & Holiday Dashboard
A simple dashboard that displays live weather, time, and Namibian public holidays.
Problem statement
Users require a quick, one-page location where they can consult instant local weather and 
context-driven data (forecast, alerts, air quality) without having to switch between various 
platforms or distrust remote/globalised information.
Main features
• Local time display
• Live weather via API
• Public holidays from JSON file
• Countdown to the weekend widget
Tech stack
• HTML for the structure,
• CSS for styling,
• JavaScript to make the webpage interactive,
• APIs are used to fetch real-time location-based data for the weather.
System requirements
Any modern web browser (Chrome, Edge, Firefox, etc.); Visual Studio Code for development
